\# Customer Feedback Analyzer - Project Guidelines



\## Project Overview

A serverless AWS pipeline that automatically analyzes customer feedback using AI.



\## AWS Services Used

1\. Amazon S3 - Store uploaded customer feedback files

2\. AWS Lambda - Automatically trigger analysis when files are uploaded

3\. Amazon Bedrock (Meta Llama) - Analyze and categorize feedback using AI

4\. Amazon DynamoDB - Store analysis results

5\. AWS CloudFormation - Deploy all infrastructure as code



\## Business Use Case

E-commerce businesses receive thousands of customer reviews daily.

This system automatically analyzes feedback, categorizes sentiment,

and stores results for business insights.



\## Important Reminders

\- Never commit API keys or secrets to GitHub

\- Stay within $200 AWS budget

\- Keep code minimal and explainable
